Web Components

The most-desired feature of web development is the ability to create your own custom HTML elements.

Every major framework has their own version of "web components": React, Angular, Vue, etc.

But these are just faked on top of regular HTML.

Now there is a true, standards-based way to make web components.

The official web components technique makes use of several HTML technologies:

  1. Templates
  2. Shadow DOM
  3. Custom Elements
  4. Javascript Modules
